# CodeAlpha Cyber Security Internship

This repository contains the tasks I completed as part of my **Cyber Security Internship at CodeAlpha**. The internship focused on building practical, hands-on skills in network security and security awareness through real-world style projects.

## About CodeAlpha

CodeAlpha is an EdTech platform that offers virtual internship programs across domains like Web Development, Python, Data Science, Android Development, AI, and Cyber Security, giving students and early-career professionals practical project experience.

## Tasks Completed

### 1. Phishing Awareness Training
A project aimed at educating users on how to identify and avoid phishing attacks.

**What it does:**
- Demonstrates common phishing techniques (fake emails, spoofed links, urgency-based social engineering)
- Provides examples of real vs. fake emails/websites for comparison
- Includes a simple quiz/checklist to help users test their ability to spot phishing attempts
- Outlines best practices for staying safe online (verifying senders, checking URLs, avoiding suspicious attachments)

**Why it matters:**
Phishing remains one of the most common attack vectors, and this task focused on the human side of cyber security — training users to recognize and avoid social engineering attacks before they cause harm.

### 2. Network Intrusion Detection System (NIDS)
A system designed to monitor network traffic and detect suspicious or malicious activity in real time.

**What it does:**
- Captures and analyzes network packets
- Detects anomalies and known attack patterns (e.g., port scans, unusual traffic spikes, suspicious IPs)
- Logs and alerts on potential intrusions
- Provides a foundation for understanding signature-based and/or anomaly-based detection methods

**Why it matters:**
Intrusion detection is a core defensive security control. This task helped build practical understanding of how network traffic can be monitored and analyzed to catch threats early.
